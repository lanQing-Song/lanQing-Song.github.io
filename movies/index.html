那些还不错不错的电影！

每个人的品味不同，这里是我个人觉得比较优秀的一些影视，有兴趣的的朋友可以去看看。

灵笼
2019-07-13(中国大陆) / 黄莺 / 李元韬 / 陶典 / 程玉珠 / 夏一可 / 中国大陆 / 董相博 / 30分钟 / 科幻 / 动画 / 奇幻 / 惊悚 / 冒险 / 董相博 Xiangbo Dong / 汉语普通话

只有吉祥寺是想住的街道吗？ / 吉祥寺だけが住みたい街ですか？
2020-08-19 / 治愈系 日本 日剧 / ★★★★★ 力荐
一集一个小故事，每个搬家人的小故事，这剧让你体会生活的那点事，虽然是虚构的却让人觉得这是生活中真实且很平常的事，片尾加上店长的人生哲悟，让人觉得生活并没有自己觉得的那般糟。很喜欢的一部剧，很推荐喜欢日常生活哲理类剧情的朋友观看。
对不起青春 / ごめんね青春！
2020-08-19 / 校园 青春 治愈系 / ★★★★★ 力荐
为数不多的好日剧。喜欢青春校园的朋友很推荐。这剧要说主角的话该是这两男女老师哈，并不是学生恋情。很推荐。
我的事说来话长 / 俺の話は長い
2020-08-19 / 生活 日本 人生 亲情 温情 / ★★★★★ 力荐
电视剧看得很少，日剧算是是最多的了，但也两只手数得过来，尝试试看过很多剧，但大多都是看一两集就弃掉了。这里推荐的都是个人喜欢。如果你是喜欢生活类，有点哲理味的剧，那它很适合你。
一人之下 第二季
2020-08-06 / 奇幻 武侠 漫画改编 热血 动作 / ★★★★★ 力荐
如果你想看国漫，那么一人之下你一定不能错过，属于国漫中优秀的那一批，很好看，名气很大，不可错过。
茶啊二中 第四季
2020-08-06 / 青春 校园 喜剧 搞笑 爱情 / ★★★★★ 力荐
开始看这番时请不要被画风劝退，继续看下去，这番应了那句“搞笑我们是认真的！”，真的很搞笑，同时也很青春，让人不经想起曾经的校园时光，十分好看，力荐！
白兔糖 / うさぎドロップ
2020-08-06 / 治愈系 成长 温暖 温情 日本动漫 亲情 / ★★★★☆ 推荐
日常番，属于亲情，温馨，治愈系，性质如片名一样。喜欢这类的朋友很推荐。
灰与幻想的格林姆迦尔 / 灰と幻想のグリムガル
2020-08-06 / 热血 催泪 冒险 奇幻 日本 / ★★★★☆ 推荐
这部番讲的是穿越冒险的故事，有点像游戏地下城那种，但不是开挂系列，主角们很现实，很苦b的那种，很好看，但不知为啥，知名度并不高。表示很推荐。
来自风平浪静的明天 / 凪のあすから
2020-08-06 / 青春 爱情 奇幻 日本 成长 / ★★★★☆ 推荐
这番讲的真的是青春爱情修炼场，就很虐那啥。。。不过是真的好看，不是后宫番。喜欢看爱情剧的朋友推荐看这番。
玉子市场 / たまこまーけっと
2020-08-06 / 校园 治愈系 爱情 青春 / ★★★★☆ 推荐
讲的是玉子的日常故事，番剧给人的印象就是干净，纯洁。喜欢日常的朋友推荐观看，有一剧场，讲的是男女主的爱情故事，也很不错，推荐。
甲铁城的卡巴内瑞 / 甲鉄城のカバネリ
2020-08-06 / 热血 冒险 日本 科幻 / ★★★★☆ 推荐
讲的是丧尸题材，十分的热血，很精彩，推荐观看。
花田少年史
2020-08-05 / 搞笑 日本 花田一路 感动 温暖 花田少年史 / ★★★★★ 力荐
这个是真的搞笑，也有泪点，但更多的是笑点。讲的是小学生一路能看见鬼魂并且和他们之间发生的故事，其实小时候电视上就有播过。真的很好看，属于不够看系列。十分推荐。
青空 / AIR
2020-08-05 / 治愈系 催泪 温情 麻枝准 爱情 / ★★★★★ 力荐
三部曲之一的AIR（青空），同样也是有名气的作品，也许你看，但也许你听过它的主题曲：鸟之诗、鸟之诗八音盒版。同样不多说，备好纸巾快上车。
雪之少女 / Kanon
2020-08-05 / 治愈系 催泪 麻枝准 校园 青春 爱情 京阿尼 日本 / ★★★★★ 力荐
key三部曲都是由游戏改编而来，按照游戏的套路，剧情有一条主线，其余的是支线，再朝主线发展的同时过程会经历每一条支线，讲的是男主和每条线路上女主之间的故事。同样是三部曲之一的kanon，也可以叫雪之少女，同样也是名气很大的作品，隔热觉得kanon是三部曲中剧情节奏把握得最好也最完整的。个人觉得key三部曲和其他动漫给人的第一感觉就是干净，纯洁！让人觉得这才是真的让人向往的二次元世界，这才是动漫世界该有的理想样子。非常推荐，也许看完你也会有所感受。
团子大家族 / CLANNAD -クラナド-
2020-08-05 / 麻枝准 催泪 治愈系 校园 日本 爱情 青春 clannad / ★★★★★ 力荐
入坑作。玄幻+泪点+笑点+美好结局=真二次元。有时可能一首潮鸣就能让你落泪。Clannad（团子大家族）有两季，让它声名远扬的是第二季。看动漫的很少有人不知道三部曲之一Clannad的了，备好纸巾快上车。
天使的心跳 / Angel Beats!
2020-08-04 / 日本 治愈 青春 麻枝准 / ★★★★★ 力荐
算是当初的入坑作之一，玄幻+泪点+笑点+美好的结局=真二次元。
高分少女 / ハイスコアガール
2020-08-04 / 游戏 日本 街机 爱情 / ★★★★☆ 推荐
如果你喜欢玩游戏或想了解小时候常玩或常见的街机游戏的历史，那这番肯定适合你。属于好看而不腻类型。
四月是你的谎言 / 四月は君の嘘
2020-08-04 / 校园 爱情 日本 青春 / ★★★★★ 力荐
光这番名就很有格调，故事确实如名字一样带有伤感，但这是一部很积极阳光的音乐番，也是校园恋爱番，很有名气的一部作品，很推荐。
只有我不存在的城市 / 僕だけがいない街
2020-08-04 / 校园 青春 科幻 日本 / ★★★★★ 力荐
一部让人思考的番剧，有点类似东野圭吾的小说，如果你喜欢这类小说，那么这番真的很适合你，有故事有哲理。推荐！
重生计划 / ReLIFE
2020-08-04 / 青春 日本 校园 治愈系 / ★★★★☆ 推荐
再让你上一次高中，你又该如何做？很好看，推荐。
索玛丽与森林之神 / ソマリと森の神様
2020-08-04 / 治愈系 催泪 温情 奇幻 日本 / ★★★★☆ 推荐
很有童话感的一部番，感人，有泪点，画风很美。
龙与虎 / とらドラ!
2020-08-04 / 日本 青春 爱情 校园 / ★★★★★ 力荐
高分青春校园恋爱番，属于有笑点又有泪点的番，很好看，网上评价也很高，没看过的朋友很推荐去看看。
荒川爆笑团 / 荒川アンダー ザ ブリッジ
2020-08-04 / 日本 搞笑 电波 / ★★★★★ 力荐
川荒爆笑团总共两季，个人觉得是为数不多的真的能带来轻松的期待的日常搞笑和情感番。属于不够看类型。推荐。
鬼灭之刃 / 鬼滅の刃
2020-08-04 / 日本 动作 热血 冒险 /
鬼灭之刃，名气老大的一部番剧了。很精彩，推荐。
银之匙 / 銀の匙
2020-08-04 / 日本 青春 / ★★★★★ 力荐
银之匙只有两部，讲的是在一所农业学校的日常与青春烦恼，还没完结，但有生之年作者应该不会再出续集了。表示真的还没看够！！
强风吹拂 / 風が強く吹いている
2020-08-04 / 日本 青春 跑步 / ★★★★★ 力荐
这些年看了不少动漫，如果非要排个位，强风吹拂拍前三！
雾山五行
2020-08-04 / 国漫 / ★★★★★ 力荐
帧帧如插画，美得惊艳。不可多得的好作品。
